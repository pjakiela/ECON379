#### Getting Started

Start by creating a do file that downloads the data from the course website.  Your 
code will look something like this:

```

set scheme s1mono 
set more off
set seed 12345
webuse set https://pjakiela.github.io/ECON379/exercises/E7-TWFE/
webuse E7-TWFE-data.dta
```
